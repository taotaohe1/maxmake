## 🛠️ 故障排查指南

### 🔴 优先级排序
1. **ALARM类错误** - 优先处理，影响设备安全运行
2. **ERROR40-ERROR58** - 设备相关错误
3. **ERROR1-ERROR39** - G代码相关错误
4. **ERROR60-ERROR66** - SD卡相关错误
5. **ERROR71-ERROR84, ERROR253** - 其他系统错误

### 💡 常用解决方案
- 检查设备状态和连接
- 验证G代码格式正确性
- 确认参数设置范围
- 重启设备或重新初始化
- 检查硬件传感器状态
