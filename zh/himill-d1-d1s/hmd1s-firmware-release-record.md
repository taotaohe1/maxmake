# HiMill D1/D1S 固件发布记录

## 固件更新说明

### 自动更新检查

使用 MaxMakeLab 软件连接设备后，如果电脑处于联网状态，软件将自动查询最新固件版本并提示升级。

### 手动更新
具体可参考链接：[MaxmakeLab HiMill D1/D1S 固件升级](/zh/himill-d1-d1s/hmd1s-firmware-upgrade.md)
如果选择手动烧写固件，您可以按照以下步骤进行更新：

1. 打开 MaxMakeLab 软件，连接到您的 HiMill D1/D1S 设备。
2. 在软件界面中，点击“设置”按钮，打开设备设置。
3. 在设备设置中，找到“固件更新”选项。
4. 点击“打开”按钮，选择下载的最新固件。
7. 点击“烧写”按钮，完成后，重新连接设备。 

> **注意事项**：
> - 请确保在固件更新过程中保持设备供电稳定
> - 更新过程中请勿关闭设备或断开连接
