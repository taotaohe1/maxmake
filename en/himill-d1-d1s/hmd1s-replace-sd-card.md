# HiMill D1/D1S SD Card Replacement Guide: Micro SD Card Swap Tutorial

Learn how to replace the Micro SD card on [MAXMAKE HiMill D1/D1S](https://www.maxmake.com/products) CNC machine. This guide covers safety precautions, SD card removal steps, and installation procedures for file storage upgrade.

---

## Safety Precautions

> ⚠️ **🔒 Safety Operation Guidelines**
> - Disconnect power during disassembly to prevent electric shock
> - For all screws, use the short side of the wrench to loosen first, then the long side to prevent stripping
> - Save removed screws in order for easy reinstallation, take photos if possible
> {.is-warning}
