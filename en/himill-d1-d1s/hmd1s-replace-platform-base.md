# HiMill D1/D1S Platform Base Replacement Guide: Wood & Aluminum Base Swap

Learn how to replace platform base plates on [MAXMAKE HiMill D1/D1S](https://www.maxmake.com/products) CNC machine. This guide covers woodworking base and aluminum base replacement procedures with safety precautions.

---

## Safety Precautions

> ⚠️ **🔒 Safety Operation Guidelines**
> - Disconnect power during disassembly to prevent electric shock
> - For all screws, use the short side of the wrench to loosen first, then the long side to prevent stripping
> - Save removed screws in order for easy reinstallation, take photos if possible
> {.is-warning}
