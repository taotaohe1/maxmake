# レーザー加工パラメーター表

このドキュメントは、HiMillシリーズ機器によるレーザー加工の参考パラメーターを提供し、異なる材料に適切な加工パラメーターを選択して、最適な加工結果と機器の安全性を確保するのを助けます。

---

## ⚠️ 重要な注意事項

1. **使用前にレーザー使用説明書をよく読み、安全予防措置に従ってください**：
   - [レーザーモジュール使用説明](/ja/himill-d1-d1s/hmd1s-laser-module.md)

2. **鏡面加工された金属の加工は推奨されません**：レーザー反射によりレーザーモジュールが損傷する可能性があります

3. **このレーザーは青色レーザーです**：透明な材料を加工することはできません

4. **パラメーター調整の提案**：実際の結果に基づいて速度と電力を合理的に調整してください

---

## 📊 彫刻パラメーターの推奨

| 材料 | 推奨パラメーター |
|------|------------------|
| 木板 | 25~35%+2000mm/min |
| 無垢材 | 20~30%+2000mm/min |
| 非透明プラスチック | 20~30%+2000mm/min |
| 紙 | 5~10%+2000mm/min |
| ロックプレート | 25~35%+2000mm/min |
| 革 | 8~15%+2000mm/min |

> 💡 **ヒント**：材料のブランドやロットによって差異がある場合があるため、正式な加工前にスクラップ材料でパラメーターをテストすることをお勧めします。

## 📝 パラメーターの説明

- **パラメーター形式**：電力パーセンテージ + 加工速度（例：25~35%+2000mm/min）
- **電力範囲**：0-100%、材料の厚さと所望の加工効果に基づいて適切に調整してください
- **最大速度**：レーザーモードの最大加工速度は2000mm/min（約33mm/s）です
- **焦点調整**：レーザー加工の最適な焦点位置は、遮光板の底面から材料表面まで5mmです

---

## 🔒 安全使用ガイド

1. **正しい取り付け**：操作前に指示に従ってレーザーを正しく取り付けてください
2. **監視の必要性**：レーザー加工中は機械の近くに誰かがいるようにしてください
3. **禁止されている材料**：易燃性および爆発性の物体、PVCなどの塩素含有材料、ポリカーボネート、グラスファイバー、カーボンファイバー、コーティングされていない金属など
4. **材料の制限**：切断材料を積み重ねないでください（例：一度に2枚以上切断する）
5. **定期的な清掃**：加工屑を定期的に清掃し、残留屑による燃焼を防いでください
6. **高温保護**：レーザーは動作中に高温を発生します。火傷を避けるために触れないでください
7. **目の保護**：長時間裸眼でレーザースポットを観察しないでください。操作中は保護メガネを着用してください
