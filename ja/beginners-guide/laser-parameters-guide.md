# レーザー加工パラメーターガイド

このドキュメントは、レーザー加工における核心パラメーターを詳細に紹介し、初心者が各パラメーターの定義、影響、および適用場面を体系的に理解するのを助けます。これらのパラメーターの調整ルールをマスターすることで、レーザー機器をすばやく使い始め、理想的な加工結果を得ることができます。

## 📊 核心パラメーターの概要

レーザー加工の効果は複数のパラメーターによって決定されます。その中で、パワー、速度、焦点距離、波長、パルス周波数が最も重要な5つのパラメーターです。これらのパラメーターは相互に作用して協力し、特定の材料と加工ニーズに基づいて合理的にマッチングする必要があります。

## 1. パワー

### 定義

⚡ **パワーとは？**

レーザー機器が単位時間あたりに出力するレーザーエネルギーの量で、機器の「強さ」を直接反映し、ワット（W）で測定されます。デスクトップ機器のパワーは通常10Wから100Wの範囲で、パラメーター調整時に機器のソフトウェアで特定のワット数（20W、50Wなど）を直接選択することができます。

### 🔥 影響と適用

パワーは加工深さ、速度、エネルギー密度を決定する核心パラメーターであり、三者は正の相関関係にあります。パワーが高すぎると材料の燃焼や変形を引き起こし、パワーが不足すると加工効果が不十分になります。

- **加工深さ**：パワーが高いほど、レーザーエネルギーが強く、材料の溶融/気化が深くなります（例：50Wは3mmのアクリルを彫刻できますが、20Wは1mmしか彫刻できません）
- **加工速度**：同じ材料に対して、パワーが高いほど速い加工速度が可能です（強さがあるため、加工を達成しながらより速く移動できます）
- **エネルギー密度**：パワーが高いほど、レーザー焦点後の単位面積当たりのエネルギーが高くなり、材料をより簡単に貫通/深く彫刻することができます；逆に、エネルギーが分散し、加工効果が弱くなります

### 📱 デスクトップ製品の適応

デスクトップモデルは主に中小パワーを使用し、高パワーは使用せず、目的に合わせて正確にマッチングされています：

| パワー範囲 | 適用場面 | 加工能力 |
|-----------|----------|----------|
| 10W-30W   | 浅いマーキング | 金属/プラスチックのロゴ、QRコード、テキスト、薄い軟質非金属の少量の浅い彫刻 |
| 30W-60W   | 汎用 | 通常の彫刻 + 薄い材料の切断（アクリル ≤3mm、木板 ≤2mm、薄い金属板 ≤0.5mm） |
| 60W-100W  | デスクトップ高パワー | やや厚い非金属（アクリル ≤5mm、木板 ≤4mm）、一部は薄いステンレス鋼板 ≤1mmを切断可能 |

## 2. 速度

### 定義

⚡ **速度とは？**

レーザーヘッド（または加工作業台）の移動速度、つまりレーザービームが材料表面を走査する速度で、ミリメートル/秒（mm/s）で測定されます。デスクトップ機器の速度調整範囲はほとんどが0-500mm/sの間で、初心者がどこから始めればよいか分からないパラメーターの一つです。

### ⚡ 影響と適用

速度は加工効率と品質を決定し、パワーと協調しなければなりません（単一パラメーターの調整は必ず問題を引き起こします）：

- **加工時間**：速度が速いほど、レーザーが全体の加工経路を完了する時間が短くなり、効率が高くなります；速度が遅いほど、加工時間が長くなり、効率が低くなります
- **加工品質**：速度が速すぎる → レーザーが材料の単一点に滞在する時間が短すぎて、エネルギーが不足し、切断が不完全、彫刻が浅い、またはパターンがぼやけます；速度が遅すぎる → レーザーが単一点に滞在する時間が長すぎて、エネルギーが蓄積し、材料の燃焼、エッジの黒化、または変形を引き起こします
- **パワーの協調**：パワーと速度は「黄金のパートナー」です — パワーが高いほど適切に速度を上げることができ（強さがあるため、加工を達成しながらより速く移動できます）、パワーが低いと速度を下げる必要があります（強さがないため、十分なエネルギーを確保するためにより遅く移動する必要があります）

### 📱 デスクトップ製品の適応

デスクトップモデルは加工タイプによって速度を決定し、機器の制限を超えず、一般的な適応範囲：

| 加工タイプ | 推奨速度範囲 | 適用場面 |
|-----------|-------------|----------|
| レーザーマーキング | 200-500mm/s | 迅速なマーキング、深い加工不要 |
| 浅い彫刻 | 100-300mm/s | 紙、薄い革、金属表面 |
| 深い彫刻 | 30-100mm/s | アクリル、木板などの深さが必要な材料 |
| 薄い材料の切断 | 50-200mm/s | 厚い材料はより低い速度が必要 |

## 3. 焦点距離

### 定義

🎯 **焦点距離とは？**

レーザー機器の集束レンズがレーザービームを最小のスポットに集束させる最適な垂直距離（簡単に言えば：集束レンズから材料表面までの「標準距離」）で、ミリメートル（mm）で測定されます。焦点距離は集束レンズの固定された特性であり、手動で調整することはできず、作業台/レーザーヘッドを昇降させることでのみ材料表面との位置合わせを行うことができます。デスクトップ集束レンズの一般的な焦点距離は50mm、63.5mm、100mmです。

### 🔍 影響と適用

焦点距離はレーザースポットサイズ、エネルギー密度、加工精度、加工範囲を決定し、「焦点ずれ誤差」（材料表面が最適な焦点距離から外れる距離）に対する感受性が異なります：

- **短焦点距離（例：50mm）**：小さなスポットを生成し、エネルギー密度が高く、加工精度が非常に高いですが、加工範囲が小さく、焦点ずれ誤差に非常に敏感です（材料表面が1-2mm高く/低くなるだけで、スポットが大きくなり、エネルギーが分散し、加工効果が急激に低下します）
- **長焦点距離（例：100mm）**：やや大きなスポットを生成し、エネルギー密度がやや低く、加工精度がやや低下しますが、加工範囲が大きく、焦点ずれ誤差にあまり敏感ではありません（材料表面のわずかな高さの変動はスポットとエネルギーにほとんど影響しません）
- **中焦点距離（例：63.5mm）**：精度、エネルギー、加工範囲のバランスを取り、デスクトップ使用の「万能モデル」であり、ほとんどの加工場面に適しています

### 📱 デスクトップ製品の適応

| 焦点距離 | 適用場面 | 特徴 |
|----------|----------|------|
| 50mm     | 微細加工 | 金属の小文字マーキング、マイクロパターン彫刻、狭い隙間の薄い材料の切断、高精度が必要な場面 |
| 63.5mm   | 汎用 | マーキング、彫刻、薄い切断が可能、初心者の第一選択、集束レンズを頻繁に交換する必要なし |
| 100mm    | 大面積加工 | 全体のアクリル/木材彫刻、大サイズワークのバッチ加工、精度は日常のニーズを満たす |

## 4. 波長

### 定義

🌈 **波長とは？**

レーザーの排他的な物理的特性で、レーザービームの「周波数識別子」であり、ナノメートル（nm）で測定されます。波長はレーザー機器の核心コンポーネント（レーザー）によって決定され、手動で調整することはできません（波長は機器を選択するときに固定され、加工中も固定されたままです）。デスクトップで使用される一般的な波長は450nm（青色光）、1064nm（ファイバー）、10600nm（CO₂、つまり10.6μm）です。

### 🌈 影響と適用

波長の核心機能は材料の吸収特性（材料がレーザーエネルギーを「吸収」できるかどうか）を決定することであり、同時に明確な安全制限もあり、機器選択の核心的な基盤となっています：

- **材料の吸収特性**：レーザーエネルギーは材料に吸収された場合にのみ加工を達成することができます。異なる材料は特定の波長にのみ「敏感」です：
  - 非金属（木材、プラスチック、アクリル、革）は10600nm（CO₂）に対する吸収効率が非常に高い
  - 金属（鋼、アルミニウム、銅）は1064nm（ファイバー）、450nm（青色光）に対する吸収効率が高い
  - 450nm（青色光）は一部の薄い非金属と金属のバランスを取ることができ、「全方位波長」となっています

- **安全制限**：
  - 10600nm（CO₂）レーザーは肉眼で見ることができず、間違って触れると火傷を引き起こしやすいため、操作中に適切な保護を講じる必要があります
  - 450nm（青色光）、1064nm（ファイバー）レーザーは肉眼で見ることができ、光路を直感的に見ることができ、初心者が操作するのにより安全です

### 📱 デスクトップ製品の適応

| 波長 | 適した材料 | 特徴 |
|------|------------|------|
| 450nm（青色光） | 薄い非金属 + すべての金属 | デスクトップの「全方位モデル」、初心者に適し、機器サイズが小さく、消費電力が低い |
| 1064nm（ファイバー） | 金属 | 金属加工専用、金属マーキング/薄い切断に特化、非金属加工効果が悪い |
| 10600nm（CO₂） | 非金属 | 非金属加工の主力、非金属彫刻/切断に特化（深さと効率は青色光よりはるかに高い）、金属を加工できない |

## 5. パルス周波数

### 定義

🔄 **パルス周波数とは？**

パルスレーザー機器（デスクトップ主流、連続レーザーとは異なる）にのみ適用され、レーザーが毎秒に発するレーザービームの回数を指し、ヘルツ（Hz）で測定されます。機器のソフトウェアで手動で調整することができ、デスクトップの調整範囲はほとんどが0-500kHzの間です。連続レーザー機器にはこのパラメーターはありません。

### 🔄 影響と適用

パルス周波数はレーザーのエネルギー分布、細かさ、および熱影響域（レーザー加熱によって影響を受ける材料の周囲の領域）を決定します：

- **エネルギー制御**：周波数が高い → 毎秒の発射回数が多い → レーザーの総エネルギーが大きい → 加工深さが深く、効率が高い；周波数が低い → 発射回数が少ない → 総エネルギーが小さい → 加工深さが浅いが、単一点でのエネルギーがより集中する
- **熱影響域**：周波数が低い → レーザー発射の間隔が長い → 材料が熱を放散する時間がある → 熱影響域が小さい（材料が変形しにくく、エッジが黒化しにくい）；周波数が高い → 発射間隔が短い → 熱が蓄積する → 熱影響域が大きく、精度要求の低いバッチ加工に適しています

### 📱 デスクトップ製品の適応

| 周波数範囲 | 適用場面 | 特徴 |
|------------|----------|------|
| 低周波数（20kHz-100kHz） | 微細加工 | 金属の小文字/パターンマーキング、マイクロ彫刻、薄い材料の狭い隙間の切断、熱影響域が小さく、エッジが滑らか |
| 中周波数（100kHz-300kHz） | 通常加工 | ほとんどのマーキング、彫刻、薄い切断、効率と細かさのバランス、デスクトップ一般ギア |
| 高周波数（300kHz-500kHz） | 高速バッチ加工 | 大面積のシンプルなパターン、薄い材料の高速切断、高効率、細かさ要求の低い場面に適しています |

## 6. パラメーターの協調と最適化戦略

⚙️ **パラメーターはどのように協力しますか？**

レーザー加工には「万能パラメーター」はありません — 単一のパラメーターを完璧に調整しても、理想的な結果を保証することはできません。核心は、固定された焦点距離と波長と組み合わせた、パワー、速度、周波数という3つの調整可能なパラメーターの協調です。

### 🔄 パワー-速度-周波数の三角関係

この3つは加工の「核心調整グループ」であり、それぞれの変化が他のものに影響します。調整ロジックは以下の通りです：

1. **パワーを設定**：材料の厚さ/加工深さに基づいて基本パワーを決定します（厚い材料/深い彫刻には高パワー、薄い材料/浅いマーキングには低パワーを選択）
2. **速度をマッチング**：パワーが高いほど速度を上げることができ、パワーが低いと速度を下げる必要があり、「高パワー低速燃焼」と「低パワー高速加工不足」を避けます
3. **周波数を調整**：微細加工には低周波数（熱影響域が小さい）、バッチ加工には高周波数（高効率）、通常加工には中周波数を選択します。周波数は速度と同期して調整する必要があります（高速の場合は周波数を上げて十分なエネルギーを確保；低速の場合は周波数を下げて熱の蓄積を避ける）

### 🧪 材料テストプロセス（初心者必ず実施）

異なる材料や同じ材料の異なる厚さには異なるパラメーターが必要です。完成品を加工する前に必ずサンプルをテストしてください：

1. 材料サンプルを取ります（完成品と同じ材料と厚さ、5cm×5cmで十分）
2. 焦点距離と波長を固定します（これらは固定された機器の特性であり、調整する必要はありません）
3. 「中パワー + 中速度 + 中周波数」に従って基本パラメーターを設定します（デスクトップ推奨：30W+100mm/s+200kHz）
4. 基本パラメーターでサンプル上でテスト加工を行い、効果を観察します：
   - 切断が不完全/彫刻が浅い → パワーを上げる/速度を下げる/周波数を上げる（一つを選択、初心者はパワーを上げることを優先）
   - 燃焼/変形 → パワーを下げる/速度を上げる/周波数を下げる（一つを選択、初心者はパワーを下げることを優先）
   - 精度が低い/エッジが粗い → 周波数を下げる/短焦点距離に変更（初心者は周波数を下げることを優先）
5. サンプルの効果が満足できるまで1-2回繰り返し微調整し、パラメーターを保存し、完成品の加工に直接適用します

## まとめ

レーザー加工パラメーターの核心は「どれだけ高く/低く調整するか」ではなく、「マッチング」です — パラメーターを材料と加工ニーズに正確にマッチングすることです。デスクトップ初心者には、以下の核心ポイントを覚えておいてください：

1. **固定パラメーターは調整不要**：波長（機器を選択するときに決定）、焦点距離（加工精度に応じて選択、初心者は63.5mmの一般モデルを使用）
2. **ニーズによって決定される核心パラメーター**：パワーは深さを決定、速度は効率を決定、周波数は細かさを決定
3. **調整順序が重要**：まずパワーを設定、次に速度をマッチング、最後に周波数を微調整、単一パラメーターのみを調整しない
4. **初心者必ず実施**：まずサンプルをテスト、次に完成品を加工、機器ソフトウェアのプリセットパラメーターライブラリを活用する

レーザーパラメーターの調整は複雑ではありません。初心者はスクラップ材料を使って試行錯誤し、後で異なる材料を試し、「どの材料がどのパラメーターにマッチするか」のルールをまとめ、徐々に「一目でパラメーターを決定」することができます。核心は多く練習し、多くテストし、調整を間違えることを恐れず、サンプルの試行錯誤のコストは非常に低いことです。
