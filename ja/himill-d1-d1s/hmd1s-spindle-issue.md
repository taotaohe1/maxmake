# HiMill D1/D1S主軸の問題

---

## 主軸が回転しない

### 可能性のある原因
- モータードライバの接続線が緩んでいる
- 主軸モーターの故障
- ソフトウェア設定の問題（例：機器がレーザーモードになっている）

### 解決策
1. モータードライバの信号線が緩んでいるか確認し、再接続してみてください。カバーの取り外し手順については、メインコントロールボード交換のビデオを参照してください [HiMill D1/D1S エアキャビネットコンポーネントの交換](/en/himill-d1-d1s/hmd1s-replace-air-cabinet.md)

2. ソフトウェアの主軸速度設定が合理的かどうか確認してください
3. 機器がレーザーで使用されたばかりで、現在レーザーモードになっているか確認してください
4. 問題が解決しない場合は、support@maxmake.comに連絡してください

## 主軸が起動後すぐに停止する

### 可能性のある原因
- 主軸ベアリングの故障
- モータードライバの故障
- 加工負荷が過大

### 解決策
1. 加工負荷が高すぎないか確認し、切削深さまたは送り速度を適切に低減してください
2. モータードライバの信号線が緩んでいるか確認し、再接続してみてください
3. 機器を再起動して再試行してください
4. 問題が解決しない場合は、support@maxmake.comに連絡してください
