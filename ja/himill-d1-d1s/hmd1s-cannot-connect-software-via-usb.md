# HiMill D1/D1SがUSBケーブル経由でソフトウェアに接続できない

---

## ケーブルの問題

### 可能性のある原因
- USBケーブルの損傷
- USBインターフェースの接触不良
- USBケーブルが長すぎるための信号減衰

### 解決策
1. USBケーブルに明らかな損傷や切断がないか確認します
2. 高品質のUSBデータケーブルに交換します（機器付属のケーブルまたはシールド付きUSBケーブルを推奨）
3. 機器のUSBインターフェースとコンピュータのUSBインターフェースにほこりや酸化がないか確認します
4. USBケーブルを抜き差しして確実に接続されていることを確認します
5. USBインターフェースを交換します（コンピュータの別のUSBポートを試してください）
6. 長すぎるUSB延長ケーブルの使用を避けてください。必要な場合は、信号増幅機能付きの延長ケーブルを使用してください
7. メインコントロールボードのUSB接続端子が緩んでいるか接触不良があるかどうかを確認します。後部カバーを取り外す手順については、メインコントロールボード交換のビデオを参照してください [HiMill D1/D1S エアキャビネットコンポーネントの交換](/en/himill-d1-d1s/hmd1s-replace-air-cabinet.md)

## 干渉の問題

### 可能性のある原因
- 近くに強い電磁干渉源がある（モーター、トランス、無線デバイスなど）
- USBケーブルが他の高圧線と平行して布線されている
- コンピュータのUSBポートからの電源供給が不十分
- コンピュータのUSBドライバーの異常

### 解決策
1. 機器を強い電磁干渉源（高電力モーター、電子レンジ、無線ルーターなど）から遠ざけてください
2. USBケーブルを高圧線（電源ケーブルなど）と平行に布線するのを避けてください
3. USBハブ（独立電源付き）を使用して安定した電力を供給してみてください
4. 機器のUSBインターフェースに電磁シールドカバーがあるか確認します（該当する場合）
5. コンピュータのUSBドライバーを更新または再インストールします
6. コンピュータと機器を再起動し、接続を再試行します
7. 異なる環境で機器の接続を試みて、環境干渉の問題であるかどうかを確認します
8. 問題が解決しない場合は、support@maxmake.comに連絡してください
