# MaxmakeLab HiMill D1/D1SがWiFi経由でソフトウェアに接続できない

---

## 🔌 接続の問題

### ❓ 可能性のある原因
- 機器が正しく再起動または原点に戻っていない
- WiFiアカウントまたはパスワードが正しく入力されていない
- コンピュータのIPアドレス設定エラー
- ルーターの設定が異常
- 機器のアンテナ接続が不良


### ✅ 解決策

1. **🔄 機器を再起動する**
   - 機器の電源を切ります
   - 10秒待ってから機器を再起動します
   - 再起動後、機器が原点に戻っていることを確認し、設定インターフェースに入ります
   - WiFiネットワークへの再接続を試みます


2. **🔑 WiFiアカウントとパスワードを確認する**
   - WiFiアカウントとパスワードが正しく入力されていることを確認します
   - パスワードの大文字小文字の区別と特殊文字に注意してください
   - パスワードを再入力します

3. **🌐 IPアドレス設定を確認する**
   - 「ローカルIPアドレス」のドロップダウンメニューからコンピュータのIPアドレスを選択します（通常はオプションが1つのみです。複数ある場合は、コンピュータのネットワーク設定で無線ネットワークアダプターのIPアドレスを確認してください）
   - 下のフィールドに機器のIPアドレスを入力します

4. **📡 ルーターの設定を確認する**
   - ルーターが機器をブロックしていないか確認します
   - ルーターのMACアドレスフィルタリング設定を確認します
   - ルーターのWiFi周波数帯（2.4GHz）設定を確認します [この機器は2.4GHz周波数帯のみをサポートします]
   - ルーターを再起動します

5. **📱 機器のホットスポット機能をテストする**
   - 機器をホットスポットモードに切り替えます（具体的な操作手順については、[HiMill D1/D1S ネットワーク設定](/en/himill-d1-d1s/hmd1s-network-configuration.md)の「ホットスポットモードに切り替える」セクションを参照してください）
   - コンピュータを使用して機器のホットスポットを検索して接続します
   - コンピュータがホットスポットに接続できる場合は、機器のWiFiモジュールが正常に動作していることを示します

6. **📡 機器のアンテナ接続を確認する**
   - 後部カバーを取り外し、メインコントロールボードのアンテナが外れているか緩んでいないか確認します
   - 後部カバーの取り外し手順については、メインコントロールボード交換のビデオを参照してください [HiMill D1/D1S コンポーネントキャビネットの交換](/en/himill-d1-d1s/hmd1s-replace-air-cabinet.md)
   - （以前に上部カバーを取り外した場合、分解中にアンテナが引き抜かれた可能性があります）
   - 上部カバーを開き、アンテナの接続状態を確認します
   - 上部カバーの取り外し手順については、移動軸モーター交換のビデオを参照してください [HiMill D1/D1S 移動軸モーターの交換](/en/himill-d1-d1s/hmd1s-replace-moving-axis-motor.md)、特にZ軸モーター交換のセクション。上部カバーの取り外しはZ軸モーターの交換と似ています

> ⚠️ **重要なお知らせ**：
> - 上部カバーを開くときは、アンテナを引っ張らないように非常に注意してください！！！
>

## 📶 干渉の問題

### ❓ 可能性のある原因
- 近くに強い電磁干渉源がある
- WiFi信号が遮られたり減衰したりしている
- 機器がルーターから遠すぎる

### ✅ 解決策
1. 機器を強い電磁干渉源（電子レンジ、無線ルーター、Bluetoothデバイスなど）から遠ざけてください
2. WiFi信号の障害物を減らし、機器とルーターの間に金属の障壁がないことを確認してください
3. 機器をルーターに近づけるか、WiFi信号増幅器を使用してください
4. WiFi周波数帯（2.4GHz）を切り替えてみてください。この機器は2.4GHz周波数帯のみをサポートしています。5GHz周波数帯を使用している場合は、ホットスポット接続のみが利用可能です
5. ルーターと機器を再起動し、接続を再試行します
6. 問題が解決しない場合は、support@maxmake.comにお問い合わせください
