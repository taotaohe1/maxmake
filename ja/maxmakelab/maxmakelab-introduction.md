# MaxmakeLabソフトウェアの使い方

## 📖 ソフトウェアの概要

MaxmakeLabは、HiMillシリーズCNC機器専用に設計されたインテリジェント制御ソフトウェアであり、直感的な操作インターフェースと豊富な加工機能を提供し、ユーザーが様々なクリエイティブな加工タスクを簡単に完成させるのを助けます。CNCの初心者であっても経験豊富なユーザーであっても、すぐに使い始めることができます。

このソフトウェアは、CNC加工とレーザー加工の両方のモードをサポートし、完全なデザイン、工具経路生成、加工制御機能を備えており、ユーザーにワンストップの加工ソリューションを提供します。

### インターフェース機能の分割

| エリア | 名前 | 機能説明 |
|--------|------|----------|
| 1 | メニューバー | ファイルのインポートとエクスポート、ビューサイズの調整、ソフトウェア設定、ヘルプガイド、言語切り替え、モード切り替え |
| 2 | デザインバー | 様々な種類のプリミティブ（線、長方形、円、テキスト、画像など）を追加 |
| 3 | キャンバス | 一般的にデバイスの作業エリアのサイズで、加工グラフィックのデザインと編集に使用 |
| 4 | カラーバー | プリミティブに異なる色を割り当て、区別や加工パラメータの設定を容易にする |
| 5 | デバイスバー | デバイスの選択、接続、クイックコントロール |
| 6 | パラメータ設定 | CNCモードではブランクパラメータと工具ライブラリパラメータを設定、レーザーモードでは加工パラメータを設定 |
| 7 | 工具経路の生成 | 工具経路の生成方法を選択（輪郭フライス、平面フライス、V型フライス、穴あけ、レリーフなど） |
| 8 | 工具経路リスト | 既に生成された工具経路を表示 |
| 9 | クイックコントロール | レーザーモードでのクイックコントロール機能、例えば高速移動、リセットなど |
| 10 | レイヤーバー | 異なる色のプリミティブに異なる加工パラメータを割り当て |
| 11 | シミュレーションプレビュー | 加工経路のシミュレーションとプレビュー |

## 🚀 コア機能の特徴

### 1. デザイン機能
- **基本プリミティブ描画**：線、長方形、円、テキストなど
- **画像インポート**：PNG、JPG、JPEG形式の画像インポートをサポート
- **ペンツール**：複雑なベジエ曲線を作成
- **グラフィック編集**：ノード編集、オフセット、アレイ（長方形/円形）、ブール演算など
- **グラフィックライブラリ**：豊富なプリセットグラフィックを提供
- **プラグイン拡張**：ボックス生成、ギア生成などの高度な機能をサポート

### 2. デバイス制御
- **自動接続**：ワンクリックでHiMillシリーズデバイスを検索して接続
- **手動接続**：手動でシリアルポートを選択して接続
- **デバイス設定**：デバイス関連の設定とヘルプにクイックアクセス
- **状態モニタリング**：デバイスの接続状態と操作情報をリアルタイム表示

### 3. 加工機能

#### CNC加工
- **ブランク設定**：カスタムブランクサイズ
- **工具ライブラリ管理**：工具パラメータの追加と管理
- **複数の工具経路生成**：輪郭フライス、平面フライス、V型フライス、穴あけ、レリーフなど
- **工具経路シミュレーション**：加工経路の2D/3Dプレビュー
- **精密制御**：単一ステップ制御、主軸スイッチ、工具交換など
- **座標設定**：XY0とZ0座標の柔軟な設定

#### レーザー加工
- **カラーマッピング**：異なる色が異なる加工パラメータに対応
- **レイヤー管理**：レイヤーの上へ移動/下へ移動/削除
- **加工モード**：絶対座標モードと現在位置モードをサポート
- **レーザー制御**：レーザー機能とモーター高速モードをアクティブ化

## 🛠️ 典型的な加工ワークフロー

### CNC加工ワークフロー

1. **グラフィックのデザインまたはインポート**：キャンバスにグラフィックを描画するか、外部ファイルをインポートする
2. **ブランクパラメータの設定**：パラメータバーで加工材料のサイズを設定する
3. **工具の選択**：工具ライブラリから適切な工具を選択する
4. **工具経路の生成**：工具経路生成方法を選択し、加工パラメータを設定する
5. **工具経路のプレビュー**：シミュレーションプレビューエリアで加工経路を確認する
6. **デバイスの接続**：「自動接続」をクリックするか、手動でデバイスを接続する
7. **原点復帰操作**：デバイスの電源を入れた後、必ず原点復帰操作を実行する
8. **加工の開始**：「実行」ボタンをクリックして加工を開始し、いつでも一時停止または停止することができる

### レーザー加工ワークフロー

1. **グラフィックのデザインまたはインポート**：キャンバスにグラフィックを描画するか、外部ファイルをインポートする
2. **色の設定**：異なるプリミティブに異なる色を割り当てる
3. **加工パラメータの設定**：レイヤーバーで異なる色の加工パラメータを設定する
4. **効果のプレビュー**：グラフィックと加工パラメータの設定を確認する
5. **デバイスの接続**：「自動接続」をクリックするか、手動でデバイスを接続する
6. **レーザーのアクティブ化**：「レーザーをアクティブ化」ボタンをクリックする
7. **開始位置の設定**：加工の開始位置を選択する
8. **加工の開始**：「実行」ボタンをクリックして加工を開始する

## ⌨️ 一般的なショートカット

| ショートカット | 機能 |
|---------------|------|
| Ctrl + S | ファイルを保存 |
| Ctrl + Z | 操作を元に戻す |
| Delete | 選択したプリミティブを削除 |
| Esc | 選択をキャンセル |

## ❓ よくある質問と解決策

### デバイス接続の問題
- **デバイスが見つからない**：ソフトウェアを再起動し、USBデータケーブルを再接続し、デバイスの電源が適切に入っているか確認してください
- **接続されているが制御できない**：USBデータケーブルが intact であるか確認し、データケーブルまたはUSBポートを交換してみてください

### ソフトウェア使用の問題
- **インターフェース表示が異常**：コンピュータに最新のグラフィックカードドライバーがインストールされているか確認し、モニターの解像度を調整してください

### 加工の問題
- **加工速度の調整**：加工パラメータエリアの「送り速度」パラメータを調整してください。値が大きいほど、加工速度が速くなります
- **加工経路のプレビューが正しくない**：インポートしたファイルが正しいか確認するか、加工原点をリセットしてください
